## TodoList
这段时间在学习React相关内容，最近我用react相关技术实现了一个最经典的React项目--TodoList应用。
这是我的Github项目地址:https://hhw28.github.io/React-TodoList/leanCloud


## 使用方法
```
# git clone git@github.com:hhw28/React-TodoList.git

# 安装依赖
npm install

# 开发预览
npm start

# 打包
npm run build
```
